## How to run locally

Ensure you have node <= 18 installed with npm

```
node -v
npm -v
```

Clone the repo and cd to the directory

```
git clone https://github.com/jwathika/quizit --depth 1 && cd quizit
```

Install required packages

```
npm install
```

Then start the app

```
npm run start
```
